导航菜单是每个网站上的链接列表。

每当您添加新页面时，您可能都需要更新`docs.json`。页面不会自动显示。

## 导航语法

我们的导航语法是递归的，这意味着您可以创建嵌套的导航组。在页面名称中不需要包含`.mdx`扩展名。

<CodeGroup>

```json 常规导航
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": ["quickstart"]
        }
      ]
    }
  ]
}
```

```json 嵌套导航
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": [
            "quickstart",
            {
              "group": "Nested Reference Pages",
              "pages": ["nested-reference-page"]
            }
          ]
        }
      ]
    }
  ]
}
```

</CodeGroup>

## 文件夹

只需将您的MDX文件放在文件夹中，并在`docs.json`中更新路径即可。

例如，要在`https://yoursite.com/your-folder/your-page`创建一个页面，您需要创建一个名为`your-folder`的文件夹，其中包含一个名为`your-page.mdx`的MDX文件。

<Warning>

除非将其嵌套在另一个文件夹中，否则您不能使用`api`作为文件夹名称。Mintlify使用Next.js，它保留了顶级`api`文件夹用于内部服务器调用。像`api-reference`这样的文件夹名称是可以接受的。

</Warning>

```json 带文件夹的导航
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## 隐藏页面

未包含在`docs.json`中的MDX文件不会在侧边栏中显示，但可以通过搜索栏和直接链接访问。
