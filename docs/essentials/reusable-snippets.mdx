## 创建自定义代码段

**前提条件**：您必须在 `snippets` 目录中创建代码段文件。

<Note>
  `snippets` 目录中的任何页面都将被视为代码段，不会被渲染成独立的页面。如果您想从代码段创建独立页面，请将代码段导入到另一个文件中并将其作为组件调用。
</Note>

### 默认导出

1. 在代码段文件中添加您想在多个位置重复使用的内容。您可以选择添加变量，这些变量可以在导入代码段时通过props填充。

```mdx snippets/my-snippet.mdx
你好，世界！这是我想在多个页面中重复使用的内容。我今天的关键词是 {word}。
```

<Warning>
  您想要重复使用的内容必须位于 `snippets` 目录中，导入才能正常工作。
</Warning>

2. 将代码段导入到目标文件中。

```mdx destination-file.mdx
---
title: 我的标题
description: 我的描述
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## 标题

Lorem impsum dolor sit amet.

<MySnippet word="香蕉" />
```

### 可重用变量

1. 从代码段文件中导出变量：

```mdx snippets/path/to/custom-variables.mdx
export const myName = '我的名字';

export const myObject = { fruit: '草莓' };
```

2. 从目标文件导入代码段并使用变量：

```mdx destination-file.mdx
---
title: 我的标题
description: 我的描述
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

你好，我的名字是 {myName}，我喜欢 {myObject.fruit}。
```

### 可重用组件

1. 在代码段文件中，通过以箭头函数的形式导出组件来创建接收props的组件。

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... 代码段内容 ...</p>
  </div>
);
```

<Warning>
  MDX 不会在箭头函数的主体内编译。尽可能坚持使用 HTML 语法，或者如果需要使用 MDX，请使用默认导出。
</Warning>

2. 将代码段导入到目标文件中并传入props

```mdx destination-file.mdx
---
title: 我的标题
description: 我的描述
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'自定义标题'} />
```
